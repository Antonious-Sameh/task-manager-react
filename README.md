## ✅ Features

- 🆕 Add New Tasks
- 🗑️ Delete Tasks
- ✏️ Edit Tasks (function ready to extend)
- 💾 LocalStorage Support
- 🎨 Stylish UI with Bootstrap
- ⚛️ Context API for state management
- 📱 Responsive Design

- React JS
- React Router DOM
- Bootstrap 5
- Context API
- LocalStorage
- Vercel (for deployment)
